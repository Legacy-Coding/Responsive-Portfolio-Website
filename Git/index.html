Git Cheat Sheet

  1. Git:

    2. What is Git? 
    Git is a DevOps tool used for source code management.
Git is used to tracking changes in the source code, enabling multiple developers
to work together on non-linear development.

  2. Exclude file:

    This will Exclude that file 
    touch .gitignore

  3. Initilize Git:

    To Initilize git in a folder 
    git init

  4. Add Files to Git:

    This will add all files and folders in git observation 
    git add .

  5. Add Files to Git:

    This will add only named files and folders in git observation 
    git add name

  6. To Comment:

    This will add comment along with the files 
    git commit -m "Your Commit"

  7. Status:

    To see the status of your project 
    git status

  8. Host Your project:

    This will hos your project to github just arrange url space in github and copy ssh key 
    git remote add origin (ssh url)

  9. Push Update:

    To Push the changes you made in your project 
    git push origin master

  10. Compare:

    to see the changes that has made / it compare stagging area to working directory 
    git diff

  11. Remove git:

    to remove or uninstall git tracking 
    -rm rf --a

  12. Git clone:

    to clone any repositories 
    git clone "url"

  13. Compare All:

    compare all the commit or changes while diff only show one version before 
    git diff --stage;

  14. Comment :

    to commit direct without stagging area 
    git commit -a -m  "Direct commit"

  15. Remove from git:

    to remove or delete files from git observation 
    git rm name.txt

  16. Rename:

    to Rename files and folders 
    git mv name.txt rename.txt
